Defining parameter data types in a Python toolbox

Desktop ¬ª Geoprocessing ¬ª Creating tools ¬ª Creating tools with Python ¬ª Creating tools in a Python toolbox

In a Python toolbox, the parameter's datatype is set using the Parameter class within the getParameterInfo method.

def getParameterInfo(self):
    #Define parameter definitions

    # First parameter
    param0 = arcpy.Parameter(
        displayName="Input workspace",
        name="in_workspace",
        datatype="DEWorkspace",
        parameterType="Required",
        direction="Input")

Composite data types

Occasionally, the need will arise for a parameter to accept multiple data types, often referred to as a composite data type. In a Python toolbox, composite data types are defined by assigning a list of data types to the parameter's datatype property. In the example below, a parameter is defined that will accept either a Raster Dataset or a Raster Catalog.

def getParameterInfo(self):
    #Define parameter definitions

    # First parameter
    param0 = arcpy.Parameter(
        displayName="Input Raster Dataset",
        name="in_rasterdataset",
        datatype=["DERasterDataset", "DERasterCatalog"],
        parameterType="Required",
        direction="Input")
